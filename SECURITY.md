# Security Policy

## Reporting a Vulnerability

Please report suspected vulnerabilities privately by emailing
security@covebitcoinwallet.com.

You can also use GitHub's private vulnerability reporting flow:
https://github.com/bitcoinppl/cove/security/advisories/new

Do not open a public GitHub issue for a vulnerability until disclosure has
been coordinated. Include enough detail to reproduce or understand the issue,
including the affected platform, app version or commit, and any relevant logs or
screenshots.

This address is for security reports only. For support requests, contact
support@covebitcoinwallet.com.
